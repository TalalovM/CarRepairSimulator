# AutoFix

Это готовый учебный веб-проект на HTML, CSS и JavaScript: современный симулятор покупки, диагностики, ремонта и перепродажи автомобилей.

## Что умеет приложение

- показывает тёмный dashboard в стиле AutoFix;
- адаптируется под мобильный экран;
- показывает уникальные автомобили без повторов;
- считает видимый ремонт, прогноз и реальный риск;
- позволяет купить автомобиль и отправить его в гараж;
- после диагностики открывает скрытые поломки;
- даёт ремонтировать отдельные узлы машины;
- показывает готовность машины к продаже;
- рассчитывает оценку, вложения и прибыль или убыток;
- ведёт историю сделок;
- сохраняет прогресс в `localStorage` браузера;
- позволяет начать новую игру.

## Как открыть

1. Откройте Visual Studio.
2. Выберите "Open a local folder".
3. Укажите папку `CarRepairSimulator`.
4. Откройте `index.html` в браузере.

## Основные файлы

- `index.html` - разметка страниц и шаблоны карточек.
- `style.css` - внешний вид приложения.
- `app.js` - вся логика симулятора.
- `assets` - изображения автомобилей.
