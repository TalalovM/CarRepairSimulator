# 🚗 AutoFix — Симулятор авторемонта и менеджмента СТО

**AutoFix** — это интерактивный веб-симулятор, в котором игрок примеряет на себя роль профессионального автомеханика и перекупщика. Покупайте машины на динамичном рынке, проводите глубокую диагностику систем, устраняйте видимые и скрытые поломки, повышайте свою репутацию и зарабатывайте капитал!

> **Новое в версии 3.0:** Игра переросла рамки обычного фронтенда! Теперь проект подключен к облачному бэкенду, имеет полноценную регистрацию игроков и живую онлайн-таблицу лидеров.

---

## ✨ Ключевые особенности игры

* **Реалистичный цикл СТО:** Покупка б/у автомобилей, дефектовка, ремонт отдельных узлов (двигатель, ходовая, электрика, тормоза, кузов) и стратегическая продажа на пике стоимости.
* **🔐 Облачная авторизация (Supabase Auth):** У каждого игрока теперь есть личный кабинет мастера. Ваш прогресс защищен, а войти в свою мастерскую можно с любого устройства.
* **☁️ Облачные сохранения:** Баланс тенге (₸), очки опыта (XP), уровень и репутация автоматически синхронизируются с базой данных Supabase в реальном времени.
* **🏆 Живой глобальный рейтинг:** Никаких фейковых ботов! На вкладке «Рейтинг» отображается реальный Топ-10 механиков, зарегистрированных в базе данных, отсортированный по чистой прибыли.
* **📈 Система прогрессии (РПГ-элементы):** Зарабатывайте опыт за каждый успешный ремонт и удачную сделку. Повышайте уровень, чтобы открыть доступ к редким и прибыльным автомобилям (таким как легендарные *BMW E30* или *Toyota Camry V40*).
* **🎯 Риски и аналитика:** Проводите платную диагностику, чтобы выявить скрытые дефекты. Помните: машина, которая казалась выгодной по объявлению, может увести бюджет в минус, если вовремя не проверить ЭБУ или цепь ГРМ.

## 🛠️ Стек технологий

* **Frontend:** * HTML5 (семантическая верстка, шаблоны `<template>`)
  * CSS3 (переменные, Flexbox, Grid Layout, адаптивный дизайн под мобильные устройства)
  * Vanilla JavaScript (ES6+, асинхронные запросы, работа с DOM)
* **Backend & Cloud:**
  * **Supabase (PostgreSQL):** Хранение профилей пользователей, учетных данных и аналитики.
  * **Row Level Security (RLS):** Защита данных на уровне строк (пользователь может редактировать только свой профиль).
  * **Supabase JS SDK:** Интеграция базы данных напрямую во фронтенд через CDN.
* **Hosting:** GitHub Pages.
